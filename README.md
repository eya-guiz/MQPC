# MQPC
Running scripts for implementation of Model Predictive Quantum Control

Implemented Schemes:
- Uncontrained
- Terminal Eqaulity Contraint (TEC)
- Tracking

Numerical Studies:
- Comparison of Schemes
- Runtime/convergence Analysis with varying prediction horizon lengths L
- Robustness comparison of TEC MPQC with optimal control using GRAPE (open loop)
- Runtime Comparison of MPQC algorithm using (IPOPT, GRAPE, Korotov) for optimization step
